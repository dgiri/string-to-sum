# Ruby String Parser

A Ruby utility that parses strings of comma-separated numbers and calculates their sum.

## Installation

Clone this repository:

```bash
git clone https://github.com/dgiri/string-to-sum.git
cd string-to-sum
```

## Usage

### Basic Usage

Run the demo script:

```bash
ruby test_parser.rb
```

### Testing

Run the tests using:

```bash
ruby string_parser_test.rb
```
